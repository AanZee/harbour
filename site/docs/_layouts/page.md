## Available modifiers
```
page__main--colored
```

## Structure
```
page
├── page__header (optional)
├── page__main (modifier)
├── page__footer (optional)
```

## Example HTML
```html
<body class="page">
	<header class="page__header">
		...
	</header>
	<main class="page__main">
		...
	</main>
	<footer class="page__footer">
		...
	</footer>
</body>
```
